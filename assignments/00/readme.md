# Introducción al Ensamblador RISC de ARM

## Descripción de la Actividad

En esta actividad, explorarás los conceptos básicos del lenguaje ensamblador para la arquitectura RISC de ARM. El objetivo es familiarizarte con la sintaxis y las operaciones fundamentales, desarrollando un programa sencillo que demuestre tu comprensión.

## Instrucciones

1. **Lectura Previa**: Revisa los conceptos básicos del ensamblador ARM, incluyendo registros, instrucciones aritméticas, de control y de memoria.

2. **Desarrollo del Programa**:

   - Escribe un programa en ensamblador ARM que realice las siguientes operaciones:
     - Solicite al usuario ingresar dos números enteros.
     - Realice las operaciones de suma, resta, multiplicación y división con los números ingresados.
     - Muestre los resultados de cada operación en pantalla.

3. **Comentarios y Documentación**:

   - Comenta tu código adecuadamente, explicando la función de cada sección y las instrucciones utilizadas.

4. **Entrega**:

   - La entrega debe realizarse en un **Gist de GitHub** en formato **Markdown**. El Gist debe incluir:
     - El código fuente del programa.
     - Una breve explicación de cómo funciona el programa.
     - Instrucciones sobre cómo compilar y ejecutar el código.

## Rubrica de Evaluación (100%)

- **Funcionamiento del Programa (25%)**:
  - El programa compila sin errores y ejecuta correctamente las operaciones solicitadas.

- **Uso Correcto de Instrucciones y Sintaxis (25%)**:
  - Se utilizan adecuadamente las instrucciones de ensamblador ARM y se sigue la sintaxis correcta.

- **Comentarios y Documentación (25%)**:
  - El código está bien comentado, facilitando la comprensión de las funciones y procesos.

- **Entrega y Formato (25%)**:
  - La entrega se realizó correctamente en un Gist de GitHub en formato Markdown, incluyendo el código, explicación y instrucciones de uso.
